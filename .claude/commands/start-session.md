# /start-session

Read `.claude/notes/claude-note.md` in full.

Then do the following:

1. Summarize the current project state in 2–4 sentences — milestone, branch, what's in flight.
2. List any deferred items or open threads that need attention.
3. Note any tooling or environment quirks to keep in mind.
4. Ask Lofi what we're working on today.

Do not propose a plan or start any work until Lofi responds.
