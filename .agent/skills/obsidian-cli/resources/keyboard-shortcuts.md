# Keyboard Shortcuts (TUI)

Available in the interactive TUI (`obsidian` with no arguments).

## Navigation

| Action | Shortcut |
|---|---|
| Move cursor left | `←` / `Ctrl+B` |
| Move cursor right (accepts suggestion at end of line) | `→` / `Ctrl+F` |
| Jump to start of line | `Ctrl+A` |
| Jump to end of line | `Ctrl+E` |
| Move back one word | `Alt+B` |
| Move forward one word | `Alt+F` |

## Editing

| Action | Shortcut |
|---|---|
| Delete to start of line | `Ctrl+U` |
| Delete to end of line | `Ctrl+K` |
| Delete previous word | `Ctrl+W` / `Alt+Backspace` |

## Autocomplete

| Action | Shortcut |
|---|---|
| Enter suggestion mode / accept suggestion | `Tab` |
| Exit suggestion mode | `Shift+Tab` |
| Enter suggestion mode (from fresh input) | `↓` |
| Accept first/selected suggestion (at end of line) | `→` |

## History

| Action | Shortcut |
|---|---|
| Previous history / navigate suggestions up | `↑` / `Ctrl+P` |
| Next history / navigate suggestions down | `↓` / `Ctrl+N` |
| Reverse history search (type to filter, `Ctrl+R` to cycle) | `Ctrl+R` |

## Other

| Action | Shortcut |
|---|---|
| Execute command or accept suggestion | `Enter` |
| Undo autocomplete / exit suggestion / clear input | `Escape` |
| Clear screen | `Ctrl+L` |
| Exit | `Ctrl+C` / `Ctrl+D` |
